**Group 1: Solve-First Default (PROG-01)**

Scenario S1-01: "Free-text DAX request gets immediate answer"
- Precondition: PBIP project open, .pbi-context.md exists with Model Context

Scenario S1-02: "Free-text request in paste mode"
- Precondition: No .SemanticModel directory (paste mode)
- Step 1: User types `/pbi calculate year-over-year growth`
- Expected: Skill outputs DAX for YoY growth immediately in copy-paste format.
- Pass: DAX appears without asking about tables/columns.

Scenario S1-03: "Refinement request is NOT treated as failure"
- Precondition: Skill just answered a DAX request
- Step 1: User types "make it a percentage"
- Expected: Skill refines the measure with percentage format. Does NOT escalate.
- Pass: Refined answer without escalation question.

Scenario S1-04: "Follow-up request is NOT treated as failure"
- Precondition: Skill just answered a DAX request
- Step 1: User types "can you also add a month-over-month version?"
- Expected: Skill creates the additional measure. Does NOT escalate.
- Pass: New measure without escalation question.

**Group 2: Escalation (PROG-02, PROG-03, INTR-01, INTR-02, INTR-03)**

Scenario S2-05: "Re-escalation targets next unresolved gap"

- Expected: `## Escalation State` section contains the gathered context.
- Pass: Section exists with the answered gap type and summary.
