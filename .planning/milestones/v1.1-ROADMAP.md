## Milestone Summary

**Key Decisions:**

- Use explicit `- Field:` bullet syntax in Last Command instructions — eliminates field-name ambiguity when Claude resolves names during context writes
- Hard gate must be three-branch (not two-branch) — the third branch (re-output on unmatched input) is what prevents soft-gate failure
- Gate tokens differentiated: `continue` for mid-session (non-terminal) vs `confirm` for final session close gate
- Model review operates on described context only — no file reads; sparse descriptions get a non-blocking "brief description" message
